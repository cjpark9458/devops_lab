**#1. top -bn1**
1) -b(Batch Mode) : 텍스트로 한번만 출력
2) -n1 : 1번 실행


**#2. awk '/%Cpu\(s\)/ {print "Usage: " 100-$8 "%"}'**
1) awk는 한 줄의 문장을 공백(스페이스) 기준으로 나눠서 번호를 매김
2) awk '/패턴/' : 출력 내용 중에서 해당 패턴이 포함된 줄을 출력
- awk 'CPU' : CPU라는 변수 출력
- awk '/CPU/' : CPU라는 패턴 검색
2) awk '/패턴/ { }' : 패턴을 찾고 { }안의 내용을 실행
