**#1. top -bn1**

2) -n1 : 1번 실행


**#2. awk '/^%Cpu\(s\)/ {print "Usage: " 100-$8 "%"}'**
1) awk는 한 줄의 문장을 공백(스페이스) 기준으로 나눠서 번호를 매김
2) awk '/패턴/' : 출력 내용 중에서 해당 패턴이 포함된 줄을 출력
- awk 'CPU' : CPU라는 변수 출력
- awk '/CPU/' : CPU라는 패턴 검색
3) awk '/^(패턴)/' : ^는 패턴의 시작을 의미하는 정규표현식이며, 줄의 맨 앞이 패턴으로 시작하는 경우만 찾으라는 의미
4) awk '/패턴/ { }' : 패턴을 찾고 { }안의 내용을 실행


**awk 'NR==2 {printf "[%-10s] Total: %5s | Used: %5s (%s) | Free: %5s\n", $6, $2, $3, $5, $4}**
1) NR==2 : NR(Number of Records)의 약자로, 2행을 출력하라는 의미
2) %-10s : 10칸을 확보하고 왼쪽 정렬
3) %5s : 5칸을 확보하고 오른쪽 정
